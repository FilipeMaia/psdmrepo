Welcome to the Explorer Web Application!

This application provides an access to data catalogs of LCLS experiments. The catalogs are managed by the OFFLINE Data Management System of LCLS. An actual implementation of the software is based on the iRODS File Manager. The application provides two views onto the catalogs: based on LCLS instruments and experiments, and based on the low-level catalog tree of iRODS. Most users will probably more interested in the former view. Please refer to the documentation on the application found in the Help section of the top-level menu. You can also submit bug reports, suggestions using the built-in Help.


Supported browsers: Note that this is a so called "rich" Web application which heavily uses JavaScript. Older and less advanced Web browsers may not support certain features on which the application's implementation relies upon. If you notice any problems with page rendering or the dynamic behavior please consider upgrading your browser. The application has been successfully tested against the following browsers: Firefox 3.x, Chrome, Safari 4 and Microsoft Internet Explorer 8 (in both native and compatibility modes).


Browsers' navigation buttons: Please, avoid using them for now! The future versions of the application will provide better integration with the browsers' navigation.


Problems monitoring: In order to better assist users with any problems (should they occur when using the application) the application may record selected user activities and get them stored in the database. The gathered information can be used to analyze a sequence of actions and user's input made before a problem occurs.


Where to go next?


Select the first item from the Browse menu to begin browsing through data catalogs of experiments. Note that this application won't allow seeing the actual data.
