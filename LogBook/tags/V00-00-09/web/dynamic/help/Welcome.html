Welcome to the LogBook Web Application!

The Electronic LogBook is designed to anotate data taken during the LCLS experiments. The data are stored in a database. This application allows for an interactive browsing and inserting new data into the database. The contents of the database can also be updated/modified directly by other applications from the LCLS Detector Control and LCLS OFFLINE domains. Pease refer to the documentation on the application found in the Help section of the top-level menu. You can also submit bug reports, suggestions using the built-in Help.


Supported browsers: Note that this is so called "rich" Web application which heavily uses JavaScript. Older and less advanced Web browsers may not support certain features on which the application's implementation relies upon. If you notice any problems with page rendering or the dynamic behavior please consider upgrading your browser's version or using more advanced browser. The application has been successfully tested against: Mozilla's Firefox 3, Google's Chrome, Apple's Safari 4 and Microsoft's Internet Explorer 8 in both native and compatibility modes.


Browsers' navigation buttons: Please, avoid using them for now! The future versions of the application will provide better integration with the browsers' navigation.


Problems monitoring: In order to better assist users with any problems (should they occure when using the application) selected user activities may be recorded and stored in the database. The gathered information will be used to analyze a sequence of actions and user's input made before a problem occures.


Where to go next?


Just select an experiment (your choice may be restricted based on your login account) using either select link on top of the page, or via the top-level menu.
